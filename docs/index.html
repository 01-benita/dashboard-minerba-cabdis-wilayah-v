Import GeoJSON PETI

Upload file `.geojson/.json` bertipe `FeatureCollection` atau `Feature`. Properti yang dibaca bila tersedia: `id`, `nama`, `komoditas`, `sumber`, `tanggal`, `keterangan`, dan `photoDataUrl`. Geometri `Point` akan dipakai langsung, sedangkan geometri lain akan memakai titik representatif.
Pilih file GeoJSON PETI, lalu tambahkan datanya ke peta dan tabel.

Import GeoJSON Izin

Upload file `.geojson/.json` bertipe `FeatureCollection` atau `Feature`. Gunakan properti seperti `Nama`, `Izin`, `Status`, `Komoditas`, `Kabupaten`, `Kecamatan`, `Desa`, `Area`, `Nomor_SK`, `Tanggal`, `Masa`, `RKAB`, dan `Dok_Teknis`. Geometri yang didukung: `Polygon` atau `MultiPolygon`.
Pilih file GeoJSON izin, lalu tambahkan datanya ke peta pada sesi ini.

Upload Koordinat WIUP

Upload file Excel dengan format kolom `No`, `X`, dan `Y`. Polygon hanya akan ditampilkan sebagai preview di peta dan tidak disimpan ke database atau data utama dashboard.

Upload Koordinat WPR
